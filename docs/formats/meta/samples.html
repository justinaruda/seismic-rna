Metadata for Samples

About metadata for samples

Purpose of metadata for samples

Define metadata about samples, such as the cell line or buffer in which the RNA was probed, the type and concentration of the chemical probe, and the temperature during chemical probing.

Commands that use metadata for samples

  • seismic export

Fields of metadata for samples

Name

Type

Description

Required

Sample

str

Name of the sample

yes

…

n/a

Additional metadata of the sample

no

Notes about metadata for samples

  • Fields are case-sensitive and must be in the file’s first line.

  • Additional fields with arbitrary names can be given. Their data types will be inferred as strings, integers, or floats.
