Metadata for References

About metadata for references

Purpose of metadata for references

Define metadata about reference sequences, such as the coordinates at which barcodes are located.

Commands that use metadata for references

  • seismic export

Fields of metadata for references

Name

Type

Description

Required

Reference

str

Name of the reference sequence

yes

…

n/a

Additional metadata of the reference

no

Notes about metadata for references

  • Fields are case-sensitive and must be in the file’s first line.

  • Additional fields with arbitrary names can be given. Their data types will be inferred as strings, integers, or floats.
