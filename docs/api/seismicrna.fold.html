seismicrna.fold.main.run(fasta: str, input_path: tuple[str, ...], *, sections_file: str | None = '', coords: tuple[tuple[str, int, int], ...] = (), primers: tuple[tuple[str, seismicrna.core.seq.DNA, seismicrna.core.seq.DNA], ...] = (), primer_gap: int = 2, quantile: float = 0.0, temp_dir: str = './temp', save_temp: bool = False, max_procs: int = 16, parallel: bool = True, rerun: bool = False)

Run the structure module.

Parameters

  • sections_file (str | None) – CSV file of sections by name, reference, and coordinates/primers [keyword-only, default: ‘’]

  • coords (tuple) – Reference name, 5’ end, and 3’ end of a section; coordinates are 1-indexed and include both ends [keyword-only, default: ()]

  • primers (tuple) – Reference name, forward primer, and reverse primer of a section; reverse primer must be given 5’ to 3’ [keyword-only, default: ()]

  • primer_gap (int) – Number of bases to leave as a gap between the end of a primer and the end of the section [keyword-only, default: 2]

  • quantile (float) – Quantile of mutation rates for normalization. Must be in [0, 1]. [keyword-only, default: 0.0]

  • temp_dir (str) – Where to write all temporary files [keyword-only, default: ‘./temp’]

  • save_temp (bool) – Whether to save temporary files when the program exits [keyword-only, default: False]

  • max_procs (int) – Maximum number of simultaneous processes [keyword-only, default: 16]

  • parallel (bool) – Whether to run multiple tasks in parallel [keyword-only, default: True]

  • rerun (bool) – Whether to regenerate files that already exist [keyword-only, default: False]
