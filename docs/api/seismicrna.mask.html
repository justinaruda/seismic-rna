seismicrna.mask.main.run(input_path: tuple[str, ...], *, coords: tuple[tuple[str, int, int], ...] = (), primers: tuple[tuple[str, seismicrna.core.seq.DNA, seismicrna.core.seq.DNA], ...] = (), primer_gap: int = 2, sections_file: str = '', count_del: bool = False, count_ins: bool = False, discount_mut: tuple[str, ...] = (), exclude_polya: int = 5, exclude_gu: bool = True, exclude_pos: tuple[tuple[str, int], ...] = (), min_finfo_read: float = 0.95, max_fmut_read: int = 0.05, min_mut_gap: int = 3, min_ninfo_pos: int = 1000, max_fmut_pos: float = 0.5, max_procs: int = 16, parallel: bool = True, rerun: bool = False) → list[pathlib.Path]

Run the mask command.

Parameters:

  • coords (tuple) – Reference name, 5’ end, and 3’ end of a section; coordinates are 1-indexed and include both ends [keyword-only, default: ()]

  • primers (tuple) – Reference name, forward primer, and reverse primer of a section; reverse primer must be given 5’ to 3’ [keyword-only, default: ()]

  • primer_gap (int) – Number of bases to leave as a gap between the end of a primer and the end of the section [keyword-only, default: 2]

  • sections_file (str) – CSV file of sections by name, reference, and coordinates/primers [keyword-only, default: ‘’]

  • count_del (bool) – Whether to count deletions as mutations [keyword-only, default: False]

  • count_ins (bool) – Whether to count insertions as mutations [keyword-only, default: False]

  • discount_mut (tuple) – Do not count a specific type of mutation [keyword-only, default: ()]

  • exclude_polya (int) – Exclude stretches of consecutive A bases of at least this length. If 0, exclude none. [keyword-only, default: 5]

  • exclude_gu (bool) – Exclude positions with G and U bases (which DMS methylates very weakly at physiological pH). [keyword-only, default: True]

  • exclude_pos (tuple) – Exclude arbitrary positions, given as (reference, position). [keyword-only, default: ()]

  • min_finfo_read (float) – Filter reads with less than this fraction of informative positions. [keyword-only, default: 0.95]

  • max_fmut_read (int) – Filter reads with more than this fraction of mutated positions. [keyword-only, default: 0.05]

  • min_mut_gap (int) – Filter out reads with any pair of mutations that are separated by fewer than this number of non-mutated bases. If 0, filter out no reads. [keyword-only, default: 3]

  • min_ninfo_pos (int) – Filter positions with less than this number of informative reads. [keyword-only, default: 1000]

  • max_fmut_pos (float) – Filter positions with more than this fraction of mutated reads. [keyword-only, default: 0.5]

  • max_procs (int) – Maximum number of simultaneous processes [keyword-only, default: 16]

  • parallel (bool) – Whether to run multiple tasks in parallel [keyword-only, default: True]

  • rerun (bool) – Whether to regenerate files that already exist [keyword-only, default: False]
