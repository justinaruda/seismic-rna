Simulate Relation Vectors Module

seismicrna.sim.relate.sim_relvecs(refseq: seismicrna.core.seq.DNA, reads: pandas.core.indexes.base.Index | numpy.ndarray | list | int, ploq: pandas.core.series.Series, pmut: pandas.core.series.Series, ptri: float = 0.2, ptrc: float = 0.4, ptrn: float = 0.3)

Simulate relation vectors.

Parameters

  • refseq (DNA) – Sequence of the reference.

  • reads (pd.Index | np.ndarray | list | int) – Names of the reads or number of reads (if int).

  • ploq (Series) – Probability that each position is low-quality.

  • pmut (Series) – Mutation rate of each position.

  • ptri (float = 0.2) – Probability that a mutation is a transition.

  • ptrc (float = 0.4) – Probability that a mutation is a complementary transversion.

  • ptrn (float = 0.3) – Probability that a mutation is a non-complementary transversion.
