seismicrna package

SEISMIC-RNA Initialization Module

Expose the sub-packages demult, align, relate, cluster, and table, plus the __version__ attribute, at the top level so that they can be imported from external modules and scripts in either of the following manners:

>>> import seismicrna
>>> seismicrna.__version__

or

>>> from seismicrna import __version__

SEISMIC-RNA Main Module

This module is the entry point for the command line interface. Running

$ seismic [OPTIONS] command [OPTIONS] [ARGS]

calls the function cli() defined in this module.

Parameters:

  • out_dir (str) – Where to output all finished files [keyword-only, default: ‘./out’]

  • temp_dir (str) – Where to write all temporary files [keyword-only, default: ‘./temp’]

  • save_temp (bool) – Whether to save temporary files when the program exits [keyword-only, default: False]

  • rerun (bool) – Whether to regenerate files that already exist [keyword-only, default: False]

  • max_procs (int) – Maximum number of simultaneous processes [keyword-only, default: 16]

  • parallel (bool) – Whether to run multiple tasks in parallel [keyword-only, default: True]

  • fastqs (tuple) – FASTQ files of single-end reads [keyword-only, default: ()]

  • fastqi (tuple) – FASTQ files of paired-end reads interleaved in 1 file [keyword-only, default: ()]

  • fastqp (tuple) – FASTQ files of paired-end reads separated into 2 files [keyword-only, default: ()]

  • phred_enc (int) – Phred score encoding in FASTQ/SAM/BAM files [keyword-only, default: 33]

  • demulti_overwrite (bool) – desiginates whether to overwrite the grepped fastq. should only be used if changing setting on the same sample [keyword-only, default: False]

  • demult_on (bool) – Whether to run demultiplexing [keyword-only, default: False]

  • parallel_demultiplexing (bool) – Whether to run demultiplexing at maximum speed by submitting multithreaded grep functions [keyword-only, default: False]

  • clipped (int) – Designates the amount of clipped patterns to search for in the sample, will raise compution time [keyword-only, default: 0]

  • mismatch_tolerence (int) – Designates the allowable amount of mismatches allowed in a string and still be considered a valid pattern find. will increase non-parallel computation at a factorial rate. use caution going above 2 mismatches. does not apply to clipped sequences. [keyword-only, default: 0]

  • index_tolerance (int) – Designates the allowable amount of distance you allow the pattern to be found in a read from the reference index [keyword-only, default: 0]

  • barcode_start (int) – index of start of barcode [keyword-only, default: 0]

  • barcode_length (int) – length of barcode [keyword-only, default: 0]

  • dmfastqs (tuple) – Demultiplexed FASTQ files of single-end reads [keyword-only, default: ()]

  • dmfastqi (tuple) – Demultiplexed FASTQ files of paired-end reads interleaved in one file [keyword-only, default: ()]

  • dmfastqp (tuple) – Demultiplexed FASTQ files of mate 1 and mate 2 reads [keyword-only, default: ()]

  • fastqc (bool) – Whether to check quality of FASTQ files [keyword-only, default: True]

  • qc_extract (bool) – Whether to unzip FASTQC reports [keyword-only, default: False]

  • cut (bool) – Whether to trim reads with Cutadapt before alignment [keyword-only, default: True]

  • cut_q1 (int) – Phred score for read 1 quality trimming [keyword-only, default: 25]

  • cut_q2 (int) – Phred score for read 2 quality trimming [keyword-only, default: 25]

  • cut_g1 (tuple) – 5’ adapter for read 1 [keyword-only, default: (‘GCTCTTCCGATCT’,)]

  • cut_a1 (tuple) – 3’ adapter for read 1 [keyword-only, default: (‘AGATCGGAAGAGC’,)]

  • cut_g2 (tuple) – 5’ adapter for read 2 [keyword-only, default: (‘GCTCTTCCGATCT’,)]

  • cut_a2 (tuple) – 3’ adapter for read 2 [keyword-only, default: (‘AGATCGGAAGAGC’,)]

  • cut_o (int) – Minimum overlap of read and adapter [keyword-only, default: 6]

  • cut_e (float) – Error tolerance for adapters [keyword-only, default: 0.1]

  • cut_indels (bool) – Whether to allow indels in adapters [keyword-only, default: True]

  • cut_nextseq (bool) – Whether to trim high-quality Gs from 3’ end [keyword-only, default: False]

  • cut_discard_trimmed (bool) – Whether to discard reads in which an adapter was found [keyword-only, default: False]

  • cut_discard_untrimmed (bool) – Whether to discard reads in which no adapter was found [keyword-only, default: False]

  • cut_m (int) – Discard reads shorter than this length after trimming [keyword-only, default: 20]

  • bt2_local (bool) – Whether to perform local or end-to-end alignment. Use local with fragmented samples, on which end-to-end yields false positives at read ends. Use end-to-end with amplicon-based samples, on which local yields false negatives at read ends. [keyword-only, default: True]

  • bt2_discordant (bool) – Whether to output discordant alignments [keyword-only, default: False]

  • bt2_mixed (bool) – Whether to align individual mates of unaligned pairs [keyword-only, default: False]

  • bt2_dovetail (bool) – Whether to treat dovetailed mate pairs as concordant [keyword-only, default: False]

  • bt2_contain (bool) – Whether to treat nested mate pairs as concordant [keyword-only, default: True]

  • bt2_unal (bool) – Whether to output unaligned reads [keyword-only, default: False]

  • bt2_score_min_e2e (str) – Minimum score for a valid end-to-end alignment [keyword-only, default: ‘L,-1,-0.5’]

  • bt2_score_min_loc (str) – Minimum score for a valid local alignment [keyword-only, default: ‘L,1,0.5’]

  • bt2_i (int) – Minimum fragment length for valid paired-end alignments [keyword-only, default: 0]

  • bt2_x (int) – Maximum fragment length for valid paired-end alignments [keyword-only, default: 600]

  • bt2_gbar (int) – Minimum distance of a gap from end of a read [keyword-only, default: 4]

  • bt2_l (int) – Seed length [keyword-only, default: 12]

  • bt2_s (str) – Seed interval [keyword-only, default: ‘L,1,0.1’]

  • bt2_d (int) – Maximum number of failed seed extensions [keyword-only, default: 4]

  • bt2_r (int) – Re-seed up to this many times [keyword-only, default: 2]

  • bt2_dpad (int) – Width of padding on alignment matrix, to allow gaps [keyword-only, default: 2]

  • bt2_orient (str) – Valid orientations of paired-end mates [keyword-only, default: ‘fr’]

  • min_mapq (int) – Ignore alignments whose mapping quality is less than this number [keyword-only, default: 40]

  • min_phred (int) – Minimum Phred score to use a base call [keyword-only, default: 25]

  • min_reads (int) – Do not analyze datasets with fewer than this minimum number of reads [keyword-only, default: 1000]

  • ambrel (bool) – Whether to find and label all ambiguous insertions and deletions (improves accuracy but runs slower) [keyword-only, default: True]

  • batch_size (float) – Target size of each batch of relation vectors, in millions of base calls [keyword-only, default: 64.0]

  • coords (tuple) – Reference name, 5’ end, and 3’ end of a section; coordinates are 1-indexed and include both ends [keyword-only, default: ()]

  • primers (tuple) – Reference name, forward primer, and reverse primer of a section; reverse primer must be given 5’ to 3’ [keyword-only, default: ()]

  • primer_gap (int) – Number of bases to leave as a gap between the end of a primer and the end of the section [keyword-only, default: 2]

  • sections_file (str) – CSV file of sections by name, reference, and coordinates/primers [keyword-only, default: ‘’]

  • count_del (bool) – Whether to count deletions as mutations [keyword-only, default: False]

  • count_ins (bool) – Whether to count insertions as mutations [keyword-only, default: False]

  • discount_mut (tuple) – Do not count a specific type of mutation [keyword-only, default: ()]

  • exclude_polya (int) – Exclude stretches of consecutive A bases of at least this length. If 0, exclude none. [keyword-only, default: 5]

  • exclude_gu (bool) – Exclude positions with G and U bases (which DMS methylates very weakly at physiological pH). [keyword-only, default: True]

  • exclude_pos (tuple) – Exclude arbitrary positions, given as (reference, position). [keyword-only, default: ()]

  • min_finfo_read (float) – Filter reads with less than this fraction of informative positions. [keyword-only, default: 0.95]

  • max_fmut_read (int) – Filter reads with more than this fraction of mutated positions. [keyword-only, default: 0.05]

  • min_mut_gap (int) – Filter out reads with any pair of mutations that are separated by fewer than this number of non-mutated bases. If 0, filter out no reads. [keyword-only, default: 3]

  • min_ninfo_pos (int) – Filter positions with less than this number of informative reads. [keyword-only, default: 1000]

  • max_fmut_pos (float) – Filter positions with more than this fraction of mutated reads. [keyword-only, default: 0.5]

  • max_clusters (int) – End the clustering step after attempting this number of clusters, even if it yields the best (smallest) BIC observed thus far. If 0, do not run clustering. [keyword-only, default: 0]

  • min_nmut_read (int) – Ignore reads with less than this number of mutated positionsduring clustering. [keyword-only, default: 0]

  • em_runs (int) – Run clustering this many times for each number of clusters, randomly initializing each run. [keyword-only, default: 6]

  • min_em_iter (int) – Run every EM run for at least this number times k iterations, even if the likelihood value has already converged. [keyword-only, default: 10]

  • max_em_iter (int) – Stop every EM run after this number times k iterations, even if the likelihood value has not yet converged. [keyword-only, default: 300]

  • em_thresh (float) – Consider an EM run to have converged when the log likelihood value has increased by less than this threshold between two consecutive iterations. [keyword-only, default: 0.01]

  • rels (tuple) – Output these relationships between the read and the reference [keyword-only, default: (‘m’,)]

  • fold (bool) – Whether to predict the RNA structure using Fold [keyword-only, default: False]

  • quantile (float) – Quantile of mutation rates for normalization. Must be in [0, 1]. [keyword-only, default: 0.0]
